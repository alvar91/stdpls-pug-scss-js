## Установка
* Установите [Node.js](https://nodejs.org/en/download/)
`brew install node`
* Установите [yarn](https://yarnpkg.com/lang/en/docs/install/) 
`brew install yarn`
* Установите зависимости проекта `yarn install`

## Для разработки:
`yarn dev`

## Сборка:
`yarn build`

## Деплой на gh-pages:
`yarn deploy`
